מצפן זכויות התלמיד - חבילה לפריסה מקומית / השלמה לפריסה ב-Render/Vercel

קבצים בחבילה:
- מצפן_זכויות_התלמיד_online.html  (frontend המחובר ל-proxy)
- server.js                         (proxy שמנתב את הבקשות ל-OpenAI)
- package.json                      (תלויות וסקריפט start)

הוראות לשימוש מקומי:
1. וודא/י שיש לך Node.js מותקן (גרסה 16+).
2. חלצי את ה-zip לתיקייה במחשב.
3. בתיקייה, צרי קובץ בשם `.env` עם השורה:
   OPENAI_API_KEY=sk-....
   אופציונלי: OPENAI_MODEL=gpt-3.5-turbo
4. התקיני את התלויות והריצי את השרת:
   npm install
   npm start
5. פתחי בדפדפן: http://localhost:3000/מצפן_זכויות_התלמיד_online.html

העלאה ל-Render (קישורים מהרשמים באתר Render לשירותי Web Service):
- צרי חשבון ב-Render
- צרי New -> Web Service
- חיבור ל-GitHub repo (מומלץ) או העלאה של הקבצים באמצעות הפרויקט
- בקונפיגורציה: Build Command = npm install, Start Command = npm start
- בסעיף Environment, הוסיפי את OPENAI_API_KEY
- לאחר פריסה, Render יספק URL ציבורי שניתן לשתף.

ביטחון:
- אל תחשפי את OPENAI_API_KEY בצד הלקוח.
- השתמשי במפתח ייעודי לדמו והגבילי אותו במידת האפשר.

אם תרצי, אוכל:
- להכין עבורך ZIP מוכן (עשיתי זאת כאן) וכן הסבר מפורט לפריסה על Render שלב-אחר-שלב.
- לעזור לפרסם את הפרויקט ב-Render אם תספקי לי את אישורך לבצע פעולות (אני לא יכול לפרסם בשמך אך יכול להכין את כל הפקודות והטקסט הדרוש).
